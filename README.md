# EquationAI

Neuro-symbolic math problem solver combining LLM reasoning with deterministic SymPy computation for verified, step-by-step solutions.

## What it does

EquationAI solves math problems — from arithmetic word problems to JEE-Advanced-level integration — and shows the full step-by-step reasoning, not just a final answer. Unlike a plain "ask an LLM" approach, every computation is executed by a sandboxed SymPy engine, not generated by the model. A custom verifier cross-checks the LLM's own claimed answer against the engine's actual computed result, catching disagreements before they ever reach the user.

See [`ARCHITECTURE.md`](./ARCHITECTURE.md) for the full reasoning behind these design decisions.

## Architecture

```
User problem (natural language)
        ↓
   Gemini LLM  →  structured JSON plan (variables, steps, SymPy expressions)
        ↓
Sandboxed SymPy executor  →  runs each step, computes real results
        ↓
   Mismatch verifier  →  checks LLM's claimed answer against engine output
        ↓
   Rendered step-by-step solution (LaTeX via KaTeX)
```

## Tech stack

- **Frontend:** Next.js 16 (React 19), Tailwind CSS, KaTeX
- **Backend:** FastAPI, Python
- **LLM:** Google Gemini (`gemini-2.5-flash`) via `google.genai`
- **Computation:** SymPy, sandboxed execution (no `__builtins__` access)

## Evaluation

An automated, resumable evaluation suite covers 22 problems across 14 categories — arithmetic, algebra, unit conversion, probability, combinatorics, coordinate geometry, linear algebra, series, differential equations, and calculus including non-obvious substitution-trick integration.

**Result: 22/22** — 19 auto-verified passes, 3 manually confirmed correct (multiple equivalent valid answer forms). Includes deliberately ambiguous and nonsensical inputs to test that the system correctly recognizes when *not* to answer.

## Running locally

**Backend:**
```bash
cd backend
python -m venv venv
source venv/bin/activate  # or venv\Scripts\Activate.ps1 on Windows
pip install -r requirements.txt
python -m uvicorn app.main:app --reload --port 8000
```

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```

Requires a `GEMINI_API_KEY` in `backend/.env` (get one free at [Google AI Studio](https://aistudio.google.com)).
